# 今晚发布前回归清单

这份清单只覆盖当前版本最容易出问题、且一旦出问题就会直接影响上线体验的链路。

## 0. 发布物检查

### 服务端

- 确认服务器代码目录是 `/opt/shenliyuan`
- 确认服务端最新 commit 正确
- 确认已重新编译并重启服务

```bash
cd /opt/shenliyuan
git log -1 --oneline
cd /opt/shenliyuan/server
go build -o /opt/shenliyuan/shenliyuan ./cmd/main.go
systemctl restart shenliyuan
systemctl status shenliyuan --no-pager
```

### 客户端

- 确认 APK 来自当前代码
- 至少安装一遍最新 debug/release 包

## 1. 登录与身份

### 普通用户

- 能正常登录
- “我”页资料正常显示
- 登出后重新登录正常

### 超级管理员

- 用 `admin / admin123` 登录
- “我”页显示“超级管理员”
- 能进入“超级管理员”与“管理处”

## 2. 公告

- 首页启动后未读公告能正常弹出
- 公告列表页能正常显示
- 不出现 `/announcements/unread` 报错

## 3. 水贴

- 首页水贴列表能加载
- 发一条纯文本帖子成功
- 发一条带图片帖子成功
- 点击帖子详情，正文、评论、图片都正常显示
- 点评论按钮后才弹出输入框
- 发送评论后评论列表立即刷新

## 4. 帖子图片

- 水贴列表中的图片缩略图能显示
- 帖子详情里的大图能显示
- 点开图片预览正常
- 退出 App 再进，重复打开同一帖子图片，确认没有明显重新加载卡顿

说明：

- 当前帖子图片已经走本地磁盘缓存
- 这一步重点是确认图片 URL、详情页和缓存链路没断

## 5. 集市

- 集市首页能加载商品列表
- “曝光台”入口显示正常，不再占大版块
- 点击“曝光台”可进入完整曝光列表
- 下拉刷新时“暂无物品”组件不再变透明

### 搜索

- 搜一个完整商品名，结果应优先命中标题接近项
- 搜一个部分关键词，能模糊匹配标题/内容
- 清空搜索后，列表恢复正常

### 发帖

- 发一条出售帖子成功
- 发一条求购帖子成功
- 发一条曝光帖子成功
- 集市详情图片、价格、联系方式正常显示

## 6. 教师 / 学科榜

- 教师评分页默认能看到“学科榜”和“专业榜”
- 学科榜中能点击进入某个学科
- 学科详情页里能看到该学科下教师排名
- 提交一条教师评分后，返回榜单页平均分立刻刷新
- 新增教师时“课程名称”输入与引导文案正常

## 7. 管理员链路

### 邀请提权

- 超管邀请普通用户
- 普通用户能看到邀请
- 用户同意后直接成为管理员
- 用户重新回到“我”页后出现管理入口

### 罢免投票

- 管理员 A 发起对管理员 B 的罢免
- 待办里能看到投票项
- 另一管理员或超管投票
- 过半后管理员 B 降级为普通用户
- 管理成员列表同步更新

## 8. “我”页

- 管理人员不再直接整页展开堆在首页
- 点击“管理人员”后进入独立列表页
- 管理入口、待办入口数量显示正常

## 9. 课程提醒

### Android

- 打开课程提醒开关
- 在课前 5 分钟收到静音提醒
- 提醒不响铃、不震动
- 提醒样式为实时提醒优先路径

### iPhone

- 只验证普通静音通知
- 不验证灵动岛

## 10. 服务端接口抽查

至少抽查这些接口：

```bash
GET /api/user/profile
GET /api/posts?board=2
GET /api/admin/members
GET /api/admin/removals/pending
GET /api/announcements
GET /api/announcements/unread
GET /api/teachers
```

## 11. 日志抽查

发布后看最近日志，确认没有持续报错：

```bash
journalctl -u shenliyuan -n 100 --no-pager
```

重点看：

- 404 路由
- 400 参数错误
- 数据库迁移失败
- 图片上传失败
- 教务接口异常

## 12. 可接受的已知非阻塞项

当前版本仍存在一些历史代码告警：

- Flutter 全量 `analyze` 还有旧 warning / info
- 客户端自动化测试覆盖仍然很薄

这两项目前不阻塞上线，但不要把它们误判成“本次发布新增缺陷”。
